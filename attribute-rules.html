What do we know about attribute rules?

We’ll talk about that later, but now let’s evaluate what is an attribute and what does it stand for in html.

It is a piece of information which determines the properties of a field or tag in a database or a string of characters in a display.

In html, there many examlpes of attribute rules as in this list below:

  1. Attributes value mast have quote.
  2. Attributes are separated by spaces.
  3. Attributes need to be lowercase.
  4. Element types have private attributes.
  5. Public attributes can go on any element.
  6. No double quote inside of doule quote.
