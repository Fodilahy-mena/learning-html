List of all html rules

This is your guide

There are many valuable rules in html that you need to know properly. Now, I invite you to follow these instructions.

I made this list to make it easier for you to understand.

  1. Define a DOCTYPE at a very top.
  2. You need to define also an html lang attribute to identify the language of text content on the web.
  3. You can’t have an end tag without a start tag.
  4. If you set a starting tag, you need an end tag as well or a closing tag as it known after the content. Tags are always surrounded by angle brackets (less-than/greater-than characters). And the end tag or second tag—the switch—always starts with a forward slash.
  5. Every starting tag needs:
    • Start with a <
    • End with a >
  6. Every ending tag needs:
    • Start with a </
    • End with a >
  7. Use src attribute for an image.
  8. Void elements do not anticipate a closing tag.
  9. You should use lowercase for the element name.
  10. Do never use less-than/greater-than inside the tag and never use also double quote inside of double quote
  11. Nest your elements properly.
  12. Make sure that you indent a child element very well.
  13. Use the element as it is ment for, in case there will be a problem. In another word, there are some elements that can't be as a child of another element. For examle, element div is not allowed as a child of element label and element a is not allowed as a child of element ul in some contexts. There are a lot of them, so you need be careful.
