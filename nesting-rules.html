Definition of Nesting

Nesting is a way of putting HTML elements within other elements in the proper order. Each element must be fully contained within its parent element.

There are some rules of nesting html which help you to make a good code, so I want you to not forget this list.

  1. Put your code in the right order, in case it won't work. What I mean about this is when you've defined DOCTYPE, make sure that all of your nesting elements are inside of the html element.
  2. Define the head element as the origine of your code.
  3. In the head element, you must need a title element as its child.
  4. Each siblings must be in the same indenting.
  5. Make sure that your child element tag is closed before you close the parent element.
  6. Your code maight have many sections inside the body element, so put them in order as well. For example, header, main and footer.
  7. Becareful there are some elements called void elements that do not have a closing tag and therefore cannot have content placed inside them, so what does this mean is they make no nest.

Don't ever do this if you want a good code!.
